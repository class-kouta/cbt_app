# ドメインモデル仕様書

## 概要

心の健康をサポートするアプリのドメインモデルを定義します。
このドメインモデルは、DDD（ドメイン駆動設計）の原則に基づいて設計されています。

認知行動療法（CBT）やコーピングリストなど、心理療法機能も統合しています。

## プロダクト目的

- ストレスや不安への対処法を管理できるコーピングリスト
- 認知の歪みに気づき、適応的な思考を身につけるコラム法（CBT）

## ユビキタス言語

プロジェクトに関わる全ての人が使用する共通の用語を定義します。

### コーピング機能の用語

| 用語 | 定義 |
|------|------|
| **コーピング（Coping）** | ストレスや不安に対処するための方法やスキル |
| **コーピングリスト** | 利用可能な対処法の一覧 |
| **コーピングタグ** | コーピング方法を分類するラベル（例：身体的、認知的、社会的） |
| **お気に入りポイント** | コーピング方法の効果度を示す指標（高いほど効果的） |

### 筆記開示機能の用語

| 用語 | 定義 |
|------|------|
| **筆記開示（Writing Disclosure）** | 反芻思考を外在化するために書き出すこと |
| **反芻思考** | 頭の中で同じ考えがぐるぐる回り続けること |
| **外在化** | 内面の思考を言語化して客観視すること |

### コラム法（CBT）機能の用語

| 用語 | 定義 |
|------|------|
| **コラム法（7カラム法）** | 認知行動療法の技法の一つ |
| **状況（Situation）** | 問題となった出来事や場面 |
| **感情（Mood）** | その状況で感じた感情 |
| **自動思考（Automatic Thought）** | 自動的に浮かんだ考えやイメージ |
| **根拠（Evidence）** | 自動思考を支持する事実 |
| **反証（Counter Evidence）** | 自動思考に反する事実 |
| **適応的思考（Adaptive Thought）** | バランスの取れた合理的な考え方 |
| **現在の感情（Current Mood）** | 認知再構成後の感情 |
| **認知の歪み** | 非合理的で偏った思考パターン |
| **認知再構成** | 否定的な思考をより適応的な思考に置き換えるプロセス |

## ドメインモデル

### エンティティ

#### コーピング
**説明**: ストレスや不安に対処するための方法を表すエンティティ

**属性**:
- ID（一意識別子）
- 内容（対処法の説明）
- お気に入りポイント（効果度、デフォルト0）
- 作成日時
- 更新日時
- コーピングタグ（複数）

**振る舞い**:
- 作成
- 編集（内容とタグの更新）
- 削除
- ポイント増加（効果があった時）
- ポイント減少

#### コーピングタグ
**説明**: コーピング方法を分類するためのラベル

**属性**:
- ID（一意識別子）
- 名前
- 作成日時

**振る舞い**:
- 作成
- 編集
- 削除

#### コラム（Column）
**説明**: 認知行動療法の7カラム法の記録を表すエンティティ

**属性**:
- ID（一意識別子）
- 状況（Situation）
- 感情（Mood）
- 自動思考（Automatic Thought）
- 根拠（Evidence）
- 反証（Counter Evidence）
- 適応的思考（Adaptive Thought）
- 現在の感情（Current Mood）
- 作成日時
- 更新日時

**振る舞い**:
- 作成
- 閲覧
- 削除（編集は現時点では実装していない）

#### 筆記開示（WritingDisclosure）
**説明**: 反芻思考を外在化するための記録を表すエンティティ

**属性**:
- ID（一意識別子）
- 内容（メモ本文）
- 作成日時
- 更新日時

**振る舞い**:
- 作成
- 編集
- 削除
- 一覧表示

#### シンプルメモ帳（SimpleNotepad）
**説明**: 特定の心理療法に紐づかない、シンプルなメモを表すエンティティ

**属性**:
- ID（一意識別子）
- 内容（メモ本文）
- 作成日時
- 更新日時

**振る舞い**:
- 作成
- 編集
- 削除
- 一覧表示

#### ストレッサーとストレス反応（StressorAndResponse）
**説明**: ストレスの原因（ストレッサー）とそれに対するストレス反応を記録するエンティティ

**属性**:
- ID（一意識別子）
- ストレッサー（Stressor）：ストレスの原因となった出来事や状況
- 認知（Cognition）：頭に浮かんだ考え（自動思考）
- 気分・感情（Mood）：そのときの気持ち
- 身体反応（Body Reaction）：体に現れた変化
- 行動（Behavior）：とった行動や反応
- 作成日時
- 更新日時

**振る舞い**:
- 作成
- 編集
- 削除
- 一覧表示

### 値オブジェクト

#### コーピングコンテンツ
**説明**: コーピング方法の内容を表す値オブジェクト

**検証ルール**:
- 空文字列不可
- 最大文字数制限

#### 筆記開示コンテンツ
**説明**: 筆記開示の内容を表す値オブジェクト

**検証ルール**:
- 空文字列不可
- 最大10000文字（長文での外在化をサポート）

#### シンプルメモ帳コンテンツ
**説明**: シンプルメモ帳の内容を表す値オブジェクト

**検証ルール**:
- 空文字列不可
- 最大10000文字

### エンティティ間の関係

#### コーピング と コーピングタグ
- **関係**: 多対多
- **説明**: 1つのコーピングに複数のタグを付けることができ、1つのタグを複数のコーピングで使用できる

## 満たすべき要件

### 実装すべき機能

#### コーピングリスト機能
1. **コーピング方法の管理**
   - 作成、編集、削除、一覧表示

2. **お気に入りポイント機能**
   - ポイントの増減
   - ポイント順でのソート表示

3. **コーピングタグ機能**
   - タグの作成、一覧表示
   - タグによるコーピングの分類

#### コラム法（CBT）機能
1. **7カラム記録の管理**
   - 作成、閲覧、削除

2. **7つの項目の記録**
   - 状況、感情、自動思考、根拠、反証、適応的思考、現在の感情

3. **記録の一覧表示**
   - 過去の認知再構成記録の振り返り

#### 筆記開示機能
1. **筆記開示記録の管理**
   - 作成、編集、削除、一覧表示

2. **自由なメモ機能**
   - 反芻思考を好きなだけ書き出せる

3. **記録の振り返り**
   - 過去の筆記開示記録を閲覧

#### シンプルメモ帳機能
1. **メモ帳記録の管理**
   - 作成、編集、削除、一覧表示

2. **自由なメモ機能**
   - 各機能に当てはまらない思考をただ書き出せる

3. **記録の振り返り**
   - 過去のメモを閲覧

#### ストレッサーとストレス反応機能
1. **ストレッサーとストレス反応の管理**
   - 作成、編集、削除、一覧表示

2. **ストレス反応の4側面記録**
   - 認知（自動思考）、気分・感情、身体反応、行動

3. **記録の振り返り**
   - 過去の記録を閲覧し、ストレスパターンを把握

### 実装しない機能

#### 心理療法機能において
1. **専門的な診断や治療提案**
   - あくまでセルフケアツールとしての位置づけ

3. **強制的な通知やリマインダー**
   - ユーザーのペースを尊重するため
